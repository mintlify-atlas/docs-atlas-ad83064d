## Reporting Issues

Found a problem? Help us fix it:

<AccordionGroup>
  <Accordion title="Outdated Prompts" icon="clock">
    If you notice a prompt is outdated:
    
    1. Open an issue on GitHub with the title: `[Tool Name] prompt is outdated`
    2. Include information about the current version
    3. If possible, provide or link to the updated prompt
    4. Note when you discovered this and how you verified it
  </Accordion>
  
  <Accordion title="Incorrect Information" icon="triangle-exclamation">
    If something is wrong:
    
    1. Specify exactly what's incorrect
    2. Provide evidence or source for the correct information
    3. Suggest a correction if possible
    4. Use the tag `[Bug]` in the issue title
  </Accordion>
  
  <Accordion title="Missing Prompts" icon="plus">
    If you know of prompts that should be included:
    
    1. Open an issue with title: `Request: [Tool Name] prompts`
    2. Provide any information you have about the tool
    3. If you can contribute the prompts yourself, mention that
    4. If not, explain how they might be obtained
  </Accordion>
  
  <Accordion title="Technical Issues" icon="wrench">
    For problems with the repository itself:
    
    1. Describe what you were trying to do
    2. What happened vs. what you expected
    3. Include any error messages
    4. Mention your environment (OS, tools used, etc.)
  </Accordion>
</AccordionGroup>

## Frequently Asked Questions

<AccordionGroup>
  <Accordion title="How often is the collection updated?">
    The collection is updated regularly as new prompts are discovered or submitted. Major updates typically happen weekly, with minor additions potentially happening daily.
  </Accordion>
  
  <Accordion title="Can I request specific prompts?">
    Yes! Open an issue on GitHub requesting the prompts you're interested in. While we can't guarantee we'll find them, we'll do our best.
  </Accordion>
  
  <Accordion title="Is this legal?">
    The collection documents prompts that have been discovered through legitimate means. However, users should be aware of and respect the terms of service of individual AI tools. The collection is educational in nature.
  </Accordion>
  
  <Accordion title="Can I use these prompts in my own projects?">
    The collection is GPL-3.0 licensed. However, the legal status of using these specific prompts varies. We recommend treating them as reference material to learn from rather than directly copying them into production systems.
  </Accordion>
  
  <Accordion title="How can I verify a prompt's authenticity?">
    We do our best to verify prompts before inclusion, but you should always validate them yourself if accuracy is critical for your use case. Check the README files in each directory for information about sources and verification.
  </Accordion>
</AccordionGroup>
